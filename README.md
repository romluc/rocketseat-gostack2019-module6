# rocketseat-gostack2019-module6 :rocket:

## App built using React Native to consume github api :octocat:

### Features 👨🏻‍🚀

- Fetch repos starred by user from github, rendering through - WebViews
- Pull to refresh
- Infinite scroll
- Styled components
- Loading (Activity Indicator)
- AsyncStorage
